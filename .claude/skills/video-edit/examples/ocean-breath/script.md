# Every other breath — 10s vertical short

**Topic:** the ocean makes ~50% of Earth's oxygen (phytoplankton).
**Format:** 9:16 vertical (1080×1920), 10.0s, ocean b-roll from Pexels + text.
**Tone:** calm hook → surprising stat → quiet CTA.

## Screenplay

| t (s)   | Beat        | On screen (text)                          | Notes / motion                                  |
| ------- | ----------- | ----------------------------------------- | ----------------------------------------------- |
| 0.0–10  | b-roll      | —                                         | Pexels ocean clip, full-bleed, dark gradient    |
| 0.0–3.0 | Hook        | **Take a deep breath.**                   | fades up, gentle rise; first frame = promise    |
| 3.0–6.5 | Reveal      | Half of it came from **the ocean.**       | "the ocean" pops in lime, slight scale punch    |
| 5.0–8.0 | Stat        | **~50%** of Earth's oxygen                | big number lower-third, springs in              |
| 7.0–9.6 | Payoff/CTA  | Tiny plankton. Every other breath.        | settles in; ends on a calm landing, not a fade  |
| 10.0    | end         | —                                         |                                                 |

## Voiceover (MOSS-TTS-Nano — rendered into the clip)

Generated by avenOS's own on-device TTS (`libs/aven-ai` tts_synth, "Bella" voice),
muxed into the mp4 as `assets/vo.wav` (10.4s):

> "Take a deep breath. Half of that oxygen came from the ocean, made by plankton
> too small to see. Every other breath, is theirs."

```bash
cargo run --release --manifest-path libs/aven-ai/Cargo.toml \
  --example tts_synth --features tts -- \
  "Take a deep breath. Half of that oxygen came from the ocean, made by plankton too small to see. Every other breath, is theirs." \
  .claude/skills/video-edit/examples/ocean-breath/assets/vo.wav
```

## Build

```bash
# 1) pull a vertical ocean clip from Pexels into assets/
python3 .claude/skills/video-edit/scripts/fetch_stock.py \
  "ocean waves underwater sunlight" \
  .claude/skills/video-edit/examples/ocean-breath/assets --portrait

# 2) reference it in index.html as <video src="assets/stock_*.mp4">, then render
bash .claude/skills/video-edit/scripts/render.sh \
  .claude/skills/video-edit/examples/ocean-breath ocean-breath "Every other breath"
```
