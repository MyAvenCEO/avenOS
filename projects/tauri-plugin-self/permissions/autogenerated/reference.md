## Default Permission

Default permissions for tauri-plugin-self

#### This default permission set includes the following:

- `allow-register`
- `allow-public-key`
- `allow-unlock`
- `allow-peer-status`
- `allow-signing-public-key`
- `allow-sign`
- `allow-verify`
- `allow-lock`
