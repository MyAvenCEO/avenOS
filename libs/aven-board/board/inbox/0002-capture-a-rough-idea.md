# Example — a rough idea, captured fast

Quick capture is the whole point of the inbox. An item can be this short:

> Add keyboard shortcuts to jump between board columns.

That's enough to not lose the thought. When someone picks it up, they move it to
**plan** and flesh it out using `templates/plan.md`. Delete this example whenever
you like.
